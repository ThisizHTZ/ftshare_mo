中证1000盘中快照监控与短周期信号说明报告

基于 FTShare 实时行情快照数据生成。适用于盘中监控、异动发现和短周期统计。

1. 数据目标

本报告说明如何基于中证1000成分股的盘中实时快照数据，构造用于行情监控和短周期交易研究的信号体系。 当前数据不是逐笔成交，也不是盘口订单簿，而是通过周期性轮询实时行情快照接口得到的准高频快照序列。

每一轮快照包含中证1000成分股的最新价、涨跌幅、成交量、成交额、振幅和权重等字段。 将相邻两轮快照做差分，可以得到短周期价格变化、成交量增量和成交额增量。

3. 盘中涨速监控

盘中涨速监控用于识别短时间内价格快速上行或下行的股票。设第 t 次快照中股票 i 的最新价为 P(i,t)，上一轮快照价格为 P(i,t-1)，则相邻快照收益率为：

r(i,t) = [P(i,t) - P(i,t-1)] / P(i,t-1)

若快照间隔为 ΔT 秒，则单位时间涨速可写为：

v(i,t) = r(i,t) / ΔT

实务中可按 interval_return 从高到低排序，识别短周期买盘推动较强的股票。

  • interval_return > 0.3%：短周期快速上涨。
  • interval_return < -0.3%：短周期快速下跌。
  • 连续多个快照为正收益：可能形成盘中趋势。

4. 成交额突然放大

快照中的 turnover 是当日累计成交额，因此相邻两轮快照的成交额差分为：

ΔA(i,t) = A(i,t) - A(i,t-1)

若 turnover_delta 明显高于股票自身近期平均水平，说明该股票在当前短周期内资金活跃度突然提升。

B(i,t) = ΔA(i,t) / [avg(ΔA(i,t-m) ... ΔA(i,t-1)) + ε]

可以进一步构造量价共振信号：

S(i,t) = r(i,t) × log(1 + ΔA(i,t))

这个指标倾向于找出“价格上行且成交额同步放大”的股票。

5. 中证1000内部涨跌排名

中证1000内部涨跌排名用于观察成分股的相对强弱。可以分别按照当日涨跌幅和短周期收益率排序。

Rank_day(i,t) = rank(-change_rate(i,t))
Rank_short(i,t) = rank(-interval_return(i,t))
  • 当日涨跌排名：反映股票当天整体强弱。
  • 短周期涨跌排名：反映最近一个快照周期内的边际变化。

如果某股票当日排名一般，但短周期排名快速上升，说明它可能刚刚开始异动。

6. 短周期动量

短周期动量关注价格变化是否具有连续性。设最近 n 个快照收益率为 r(t-n+1) ... r(t)， 可以定义短周期动量为：

M(i,t,n) = Σ r(i,t-j), j = 0 ... n-1

也可以加入成交额权重，构造量价动量：

M_money(i,t,n) = Σ r(i,t-j) × log(1 + ΔA(i,t-j))

短周期动量适合捕捉盘中趋势延续、识别从弱转强的股票，并过滤无量上涨或无量下跌。

7. 异动股筛选

异动股筛选可以将多个条件组合成规则：

I(i,t) = 1[r(i,t) > θr] × 1[ΔA(i,t) > θA] × 1[change_rate(i,t) > θd]

也可以使用综合打分：

Score(i,t) = z(interval_return) + z(turnover_delta) + z(change_rate) + z(amplitude)
类型条件特征解释
放量上涨涨速高、成交额增量高资金主动推动
放量下跌跌速高、成交额增量高抛压或利空释放
高振幅活跃振幅高、成交额高多空分歧较大

8. 成分股贡献度粗估

若忽略复权、指数除数、停牌处理等细节，可以用成分股权重和涨跌幅粗略估计对指数涨跌的贡献：

C(i,t) = weight(i) × change_rate(i,t)

指数整体涨跌幅可粗略估计为：

R_index_hat(t) = Σ weight(i) × change_rate(i,t)

短周期维度也可以使用：

C_short(i,t) = weight(i) × interval_return(i,t)

该指标可以回答“当前是谁在拉动中证1000”和“当前是谁在拖累中证1000”。 需要强调的是，这只是粗估，更适合解释和监控，不适合作为精确指数复刻。

9. 推荐监控面板

  1. 涨速榜：按 interval_return 降序。
  2. 成交额增量榜：按 turnover_delta 降序。
  3. 量价共振榜：按 interval_return × log(1 + turnover_delta) 降序。
  4. 当日强势榜：按 change_rate 降序。
  5. 权重贡献榜：按 weighted_change_contribution 降序。
  6. 风险预警榜：按负向短周期收益和成交额增量组合排序。

10. 结论

中证1000盘中快照轮询数据适合用于短周期行情监控和信号发现。它可以支持涨速监控、 成交额放大识别、内部排名、短周期动量、异动股筛选和指数贡献度粗估。

但该数据源本质是快照序列，不是逐笔成交或盘口订单簿。因此，它适合做盘中监控、异动发现和短周期统计， 不适合做严格的 tick 级回测或盘口微结构研究。更稳妥的做法是：先用快照筛选异动股，再对少量异动标的拉取逐笔成交数据验证。
